
  # DormDraft

  This is a code bundle for Untitled. The original project is available at https://www.figma.com/design/VnM1aGD6VJWMmcicRR17tC/Untitled.

  ## Running the code
  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  If cloning doesn't work, run: 
* brew install git-lfs
* git lfs install



## Overview
Students sign up and are assigned a room they've never seen, buy furniture they're not sure fits, and show up on move-in day hoping it works out. DormDraft fixes that.
DormDraft is an interactive 3D isometric dorm room visualizer where students can explore their exact room type, arrange furniture to scale, price out what they need, and share their layout — all before stepping foot on campus. Select your building, pick your room type, and instantly see a true-to-scale isometric rendering of your future home. Drag in a mini fridge, a microwave stand, a reading lamp — watch the estimated cost update in real time. When you're happy with it, export it as a PNG to share with your roommate.
